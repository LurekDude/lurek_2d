```
MAJOR.MINOR.PATCH
```

| Segment   | Increment whenтАж                                                                                    |
| --------- | -------------------------------------------------------------------------------------------------- |
| **MAJOR** | Breaking API changes тАФ Lua scripts or engine configuration must be ported                          |
| **MINOR** | New backwards-compatible features тАФ new `lurek.*` APIs, new modules, new default configs           |
| **PATCH** | Bug fixes, internal refactors, documentation and tooling changes that do not affect the public API |
