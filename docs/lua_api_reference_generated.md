- `luna.load()` -- Called once after the script is loaded

- `luna.draw()` -- Called every frame for rendering
- `luna.keypressed(key)` -- Called when a key is pressed
- `luna.keyreleased(key)` -- Called when a key is released
- `luna.mousepressed(x, y, button)` -- Called when a mouse button is pressed

- `luna.touchmoved(id, x, y, dx, dy, pressure)` -- Called on touch move

Encodes a Lua table into a TOML document string. The root value must be a Lua table. Mixed string/index tables and sparse array-style tables are rejected. Empty Lua tables encode as TOML tables, so empty TOML arrays do not round-trip distinctly through plain Lua tables.
