# New Game Project

## Interview Questions
1. What genre? (platformer, top-down RPG, shooter, puzzle, visual novel, arcade, roguelike, tower defense)
2. What art style? (pixel art, smooth vector, minimalist)
3. Approximate scope? (game jam, small project, medium project)
4. Solo or team?
5. Any specific mechanics in mind?

## Workflow
1. Select template based on genre
2. Route to **game-architect** for system design
3. Route to **level-designer** for first area brief
4. Route to **lua-scripter** to scaffold from template
5. Route to **visual-artist** for colour palette + art direction

## Output
- Scaffolded project folder with conf.lua, main.lua, module structure
- System design document
- First level brief
- Color palette recommendation
