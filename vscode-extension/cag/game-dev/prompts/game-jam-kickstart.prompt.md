# Game Jam Kickstart

## Questions
1. Jam theme?
2. Time constraint? (48h, 72h, 1 week)
3. Solo or team?

## Steps
1. Scaffold game-jam template immediately
2. Route to **gameplay-designer** for 3 mechanic ideas (5 min time-box)
3. Select 1 mechanic
4. Route to **lua-scripter** to implement core loop
5. Time-box each subsequent feature
