# Luna2D Game Developer CAG Layer

AI-first workflow layer for game developers using Luna2D.

## Purpose
This is NOT for engine developers. These agents, skills, prompts, and
instructions help people MAKING GAMES with Luna2D.

## Components
- **11 Agents**: Specialized roles for game development tasks
- **26 Skills**: Domain knowledge for common game systems
- **15 Prompts**: Task-driven playbooks for common game dev workflows
- **8 Instructions**: Context rules auto-loaded by file pattern
- **12 Templates**: Project scaffolding for different game genres

## Deployment
Use `Luna: Deploy Game Dev AI Layer` command in VS Code to install
these into your game project's `.github/` folder.
