# Province Demo

Demonstrates the Luna2D province map system, inspired by Europa Universalis-style grand strategy games.

## Features Shown

- **World generation** — procedural province map via Voronoi tessellation
- **Map modes** — switch between source colour and terrain views with `M`
- **Province selection** — click to select, see terrain and owner info
- **Pathfinding** — right-click to find a path from selected province
- **Fog of war** — provinces revealed in a radius around the first province
- **Improvements** — a fort placed at the first province's centroid

## Controls

| Key / Button | Action |
|---|---|
| Left click | Select province |
| Right click | Find path from selected to clicked province |
| `M` | Cycle map mode |
