# Loot RPG

Dungeon crawler loot game built with Lurek2D.

## Controls
| Key | Action |
|-----|--------|
| Space | Enter next room |
| E | Auto-equip best gear |
| B | Buy healing potion |
| Up/Down | Scroll inventory |
| Escape | Quit |

## Features
- Item rarity system (common/rare/epic/legendary)
- Equipment slots (weapon/armor/helm/boots/accessory)
- Floor progression with scaling enemies
- Weight-limited backpack

## APIs Used
lurek.render, lurek.render_ui, lurek.input, lurek.tween, lurek.particles, lurek.camera, lurek.window, lurek.time, lurek.signal
