# Alchemy Lab

**Category:** RPG
**Engine:** Lurek2D

A potion crafting and alchemy simulation where you combine elemental ingredients at your workbench to discover and brew potions for profit.

## Gameplay

- Work at three stations: **Mortar** (grinding), **Cauldron** (heating/mixing), and **Bottle** (storing)
- Combine six unique ingredients, each with elemental properties (fire, water, earth, air)
- Manage cauldron temperature carefully — too cold yields weak potions, too hot causes explosions
- Discover recipes by experimenting with ingredient combinations
- Buy ingredients from the shop and sell brewed potions for gold
- Unlock all five recipes including the rare Philosopher's Stone

## Controls

| Key    | Action                      |
| ------ | --------------------------- |
| 1-6    | Select / add ingredient     |
| G      | Grind ingredients in mortar |
| H      | Heat cauldron               |
| J      | Cool cauldron               |
| B      | Bottle the brew             |
| S      | Toggle shop                 |
| Enter  | Start game / confirm        |
| Escape | Quit                        |

## How to Run

```bash
cargo run -- content/games/rpg/alchemy
```
