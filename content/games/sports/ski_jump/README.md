# Ski Jump

**Category:** Sports

A side-view ski jumping game with realistic approach, flight, and landing mechanics. Choose from three hill sizes, manage aerodynamics in the air, and nail the landing for maximum style points across three competitive rounds.

## How to Play

- **D** — Crouch (reduces drag, +10% speed during approach)
- **Space** — Jump at ramp end / Land at touchdown
- **W / S** — Lean forward / backward while airborne
- **1 / 2 / 3** — Select hill size: Small (90m) / Normal (120m) / Large (150m)
- **Escape** — Quit

## Features

- Three hill sizes with scaled ramp geometry
- Wind system affecting airborne trajectory
- Aerodynamic lean mechanic for optimal flight distance
- Landing style scoring with five-judge panel
- Three-round competition with combined scoring
- Particle effects: snow spray, wind lines, landing impact, crowd confetti
- Tweened HUD counters and judge score reveals

## Running

```bash
cargo run -- content/games/sports/ski_jump
```
