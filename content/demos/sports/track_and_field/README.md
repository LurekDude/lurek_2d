# Track & Field

Four classic Olympic events in one multisport game — run, jump, hurdle, and throw
your way to the best scores.

## What It Demonstrates

- `luna.keypressed()` — rapid alternating key detection for running speed
- `luna.input.isKeyDown()` — continuous power bar control
- Physics simulation — projectile arcs, hurdle clearing with gravity
- State machines — event sequencing and per-event sub-states
- UI panels — event HUD, power bar, running speed display

## Events

| # | Event | How to play |
|---|-------|-------------|
| 1 | 100m Sprint | Alternate **A** and **D** as fast as possible |
| 2 | Long Jump | Build speed with A/D, then **Space** to jump at the board |
| 3 | Hurdles | Like 100m, press **Space** to leap over each hurdle |
| 4 | Hammer Throw | Watch the spinning power bar, **Space** to release at peak |

## Controls

| Key | Action |
|-----|--------|
| A / D or Left / Right | Add speed (alternate to maintain) |
| Space | Jump / Leap / Release hammer |
| R | Restart |
| Escape | Quit |
